FORMULÁRIOS

Dados enviados para um banco de dados. Ex: →
	<form action="processa.php" method="post"> form – para formulários.  	Action – pra qual página será enviado as infos (no caso, para a processa.php). 	Method –  (POST = será criptografada e será enviada ocultamente, será 	enviada junto ao cabeçalho da página / GET= info será enviada pela url).

	<label> Text </label> Legenda para um item.
	<input type="text" name="texto1" placeholder="Eu sou um input do tipo 	texto"> input – para criar controle interativos com usuário num formulário. 	type – que tipo o input receberá. name – que identificação aquele input 	receberá. placeholder – frase que aparecerá dentro da caixa de texto.

<!-- Nem todos os inputs precisam de value, mas todos precisam dos atributos type e name. (tags inputs não são fechadas.) -->

	<br> pula uma linha

	<label>Password</label>
	<input type="Password" name="senha" placeholder="Digite uma senha">

O tipo password troca os caracteres que você está digitando por círculos ou asteriscos para que, se houver alguém do seu lado, o danadinho que fique bisbilhotando a senha cadastrada.

	<!-- placeholder = O texto aparece dentro da caixa →

	<br>

	<label>Oculto</label>
	<input type="hidden" name="informacao_oculta" value="mensagem oculta">
	O atributo hidden é um atributo booleano que indica se um elemento é 	ou 	não relevante. Por exemplo, ele pode ser usado para esconder elementos de 	página que não podem ser usados até que o processo de login seja completo.

	<!-- O input acima não aparecerá -->

	<label>Select</label> Controle que apresenta um menu de opções.

		<option value="1">Opção 1</option>
		<option value="2">Opção 2</option>
		<option value="3">Opção 3</option>
	</select>

	<br>

	<label>Radio</label> Um botão de escolha.
	<input type="Radio" name="radio1" value="1">Valor 1
	<input type="Radio" name="radio1" value="2">Valor 2

	<br>

	<label>Checkbox</label> Caixa de marcação.
	<input type="Checkbox" name="ck1" value="1" checked>Valor 1
	<input type="Checkbox" name="ck1" value="2">Valor 2
	<input type="Checkbox" name="ck1" value="3">Valor 3

 O atributo "checked" serve para forçar o usuário a marcar aquele box, pois quando o formulário for aberto pela primeira vez, este já aparecerá marcado.

	<input type="submit" name="submit" value="Enviar">
